# 🚦 UK Road Accident Data Analysis

## 📌 Project Overview
This project analyzes UK road accident data to identify patterns, trends, and insights using Python.

The main goal of this project is to:
- Understand accident trends
- Analyze accident severity
- Study location-based accident distribution
- Perform data cleaning and visualization

---

## 📂 Dataset
The dataset contains information about:
- Accident severity
- Number of vehicles involved
- Weather conditions
- Road surface conditions
- Time and location of accident

Due to GitHub file size limitations (100MB max), the dataset is not included in this repository.
You can download the dataset from here:

- Dataset: [Download Here](https://www.kaggle.com/datasets/devansodariya/road-accident-united-kingdom-uk-dataset)

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Analysis Performed
- Data Cleaning
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Visualization of accident trends
- Severity distribution analysis

---

## 📈 Key Insights
- Most accidents occur during peak traffic hours
- Weather conditions affect accident frequency
- Urban areas show higher accident counts
